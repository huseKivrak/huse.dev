## Configuration Notes

### Temperature Settings
- **Low (0.2-0.4):** For consistent, reliable characters (void, Dr. River)
- **Medium (0.5-0.7):** For balanced personalities (Corporate, Oracle)
- **High (0.8-1.0):** For chaotic, energetic characters (N30N-DR34M, jazzbox, sparkle)

### Stability Settings
- **Low (0.2-0.4):** Creates glitchy, unpredictable speech (Nyx, jazzbox, N30N-DR34M)
- **Medium (0.5-0.7):** Balanced variation (sparkle, Oracle, Captain)
- **High (0.8-0.9):** Steady, consistent delivery (void, Corporate, Dr. River)

### Speed Guidelines
- **Slow (0.7-0.9):** Contemplative characters (void, Oracle, Dr. River)
- **Normal (0.95-1.05):** Most characters
- **Fast (1.1-1.2):** Energetic characters (N30N-DR34M, sparkle, Corporate)

### Implementation Tips
1. Test each voice configuration with sample text before deploying
2. Adjust settings based on user feedback
3. Consider creating voice presets for quick switching
4. Monitor conversation logs to fine-tune personality delivery
5. Remember that voice settings can dramatically affect character perception
