elevenlabs agent configuration tool

instructions:

  1. Click on any agent card to expand its configuration
  2. Click "copy" buttons to copy individual settings
  3. Go to ElevenLabs Dashboard ‚Üí Create New Agent
  4. Paste each configuration into the appropriate field
  5. For Voice ID: Go to Voice Library ‚Üí Search for the voice ‚Üí Copy its ID
  6. Save your agent and note its Agent ID
  7. Update the NEXT_PUBLIC_ELEVENLABS_AGENT_ID in your .env file
